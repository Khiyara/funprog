Review Satu Bulan Belajar Pemrograman Fungsional

Halo, Selamat Siang semuanya, saya Rizkhi Pramudya Hastiputra, saat ini sedang menuntut ilmu di Fakultas Ilmu Komputer Universitas Indonesia jurusan Ilmu Komputer. Di semester ini saya mengambil mata kuliah Pemrograman Fungsional. Ada beberapa alasan mengapa saya mengambil mata kuliah ini, yang pertama karena saya ingin memperdalam kemampuan saya dalam dunia pemrograman dan meningkatkan skill saya, yang kedua karena jika saya mengambil mata kuliah ini, hari jumat saya jadi hari libur. Saya berniat menggunakan ilmu ini sebaik-baiknya agar berguna bagi banyak orang di kedepannya.

Di awal minggu pertemuan, saya mempelajari konsep dasar pemrograman fungsional, yaitu lambda kalkulus. Pada mata kuliah ini kami diajarkan menggunakan bahasa pemrograman Haskell. namun tidak diwajibkan untuk menggunakan Haskell, karena yang penting dari mata kuliah ini adalah paradigmanya.

Pertama didunia pemrograman terdapat 2 paradigma, yaitu Pemrograman Imperatif dan Pemrograman Deklaratif, dikebanyakan kuliah saya sebelumnya saya diajarkan bagaimana cara kerja Pemrograman Imperatif. Yang menjadi poin penting perbedaan Pemrograman Deklaratif dengan Imperatif adalah, jika Imperatif kita mendefinisikan step by step bagaimana cara mendapatkan suatu hasil, di Deklaratif kita mendeklarasikan atau mendefinisikan suatu hasil tersebut. Jadi kita tidak menjelaskan bagaimana step by stepnya.

Sebagai contoh kita ingin menghitung jumlah elemen dari list di paradigma Imperatif maka kurang lebih hasil codenya sebagai berikut:

array = [1,2,3,4,5,6] length = 0 for x in array: length += 1 print (length)

dan dengan menggunakan paradigma Deklaratif hasil codenya sebagai berikut:

def length(xs): return 0 if xs == [] else 1 + length(xs[1:]) print(length([1,2,3]))

Jika dibandingkan di Imperatif kita lebih memberikan statement step by step apa yang harus dilakukan untuk menghitung jumlah elemen sedangkan di Deklaratif kita lebih memperhatikan apa itu sebenarnya jumlah elemen, lalu kita definisikan jumlah elemen itu, yang dimana definisi dari jumlah elemen itu adalah penjumlahan 1 dengan banyaknya elemen list

Setelah itu minggu selanjutnya, kami diajar oleh Dosen yang didatangkan dari Portugal. Tetapi selama minggu ini perhatian saya kurang fokus terhadap materi yang disampaikan. Karena menurut saya materi yang disampaikan merupakan review dari mata kuliah Teori Bahasa dan Automata. Sehingga cukup mendengarkan sekilas saya sudah dapat maksud dari yang ingin dijelaskan.

Di minggu selanjutnya saya mempelajari apa itu High Order Function, menurut definisi High Order Function adalah Function yang menerima suatu fungsi sebagai parameter atau suatu fungsi yang mereturn fungsi.

Contoh umum dari High Order Function adalah Map dan Reduce, kedua fungsi tersebut sudah saya ketahui sebelum saya mengambil mata kuliah ini. Saya mengetahui kedua fungsi tersebut dijavascript. Sekarang akan dijelaskan apa itu Map dan Reduce.

Map adalah fungsi yang menerima dua argumen yaitu fungsi X dan list atau sequence Y. lalu hasil return dari Map adalah sebuah array baru atau list baru yang setiap nilai dari array baru tersebut merupakan hasil petaan dari list Y terhadap fungsi X.

Reduce adalah fungsi yang menerima 3 argumen (argumen ketiga merupakan argumen optional), yang pertama yaitu fungsi X, kedua adalah list atau sequence Y, yang ketiga adalah nilai awal (optional). Hasil dari return Reduce adalah sebuah nilai yang merupakan hasil operasi dari fungsi X terhadap setiap elemen dari list Y.

Berikut akan diberikan contoh terhadap dua fungsi map dan reduce. fungsi map di Pemrograman Fungsional kurang lebih bentuknya seperti ini : def map(func, xs): return [] if xs == [] else [func(xs[0])] + map(func, xs[1:]) multiplyBy2 = lambda x : x * 2 print(map(multiplyBy2, [1,2,3,4,5]))

yang dilakukan fungsi tersebut adalah, untuk setiap elemen di xs, ambil nilai awal array, lalu kalikan dengan 2, buat array baru yang berisi 2 dan konkatenasi dengan sisanya.

Untuk fungsi reduce di Pemrograman Fungsional kurang lebih bentuknya adalah : def fold(func, xs): return 0 if xs == [] else func(xs[0]) + fold(func, xs[1:]) multiply = lambda x,y : x*y print(fold(itself, [1,2,3,4,5]))

yang dilakukan fungsi tersebut adalah untuk setiap elemen di xs, ambil satu nilai awal array, kemudian lakukan operasi '+' untuk setiap elemen didalam list.
